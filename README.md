# EncriptarTextoAlura
Encriptador y desencriptador de texto - Desafio

Proyecto creado a partir de JavaScript, HTML y CSS.

```
"e" -> "enter"
"i" -> "imes"
"o" -> "ober"
"a" -> "ai"
"u" -> "ufat"
```

Y viceversa dependiendo de los campos y botones que oprimas.
